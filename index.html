Nxtdroid

Android based Lego Mindstorms NXT Direct Protocol implementation

    This is an Android app (and API that you can build from) that controls a Lego Mindstorms NXT via Bluetooth.

    You can generate a Codenvy clone of this project. Compile it in browser, download it directly to your Android device.
